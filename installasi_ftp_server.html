INSTALLASI FTP SERVER PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Untuk menginstall FTP ketikan perintah apt-get install proftpd setelah itu akan muncul dua pilihan, pilihlah standalone

Langkah 3

Setelah menginstall kita akan buat sebuah user yang akan di gunakan sebagai tempat untuk menyimpan file adduser ftpudin username bisa disesuaikan sesuka kalian

Langkah 4

Setelah itu restart network dan ftp dengan cara

service networking restart

service proftpd restart

Langkah 5

Kita akan menguji coba FTP dengan menggunakan aplikasi WinSCP

Langkah 6

Setelah itu buka WinSCP lalu pada menu file protocol ganti dengan ftp, kemudian di menu hostname ketikan ip kalian, untuk menu username gunakan username yang telah kita buat

Langkah 7

Installasi dan konfigurasi FTP telah berhasil
