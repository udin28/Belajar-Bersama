INSTALLASI FILE SERVER (SAMBA) PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Buatlah folder user dan share pada direktory home dengan menggunakan perintah mkdir user dan mkdir share

Langkah 3

Lalu ubah hak akses untuk folder user dan share agar dapat di akses secara full dengan menggunakan perintah chmod 777 user dan chmod 777 share

Langkah 4

Setelah itu kita akan menginstall file server/samba dengan menggunakan perintah apt-get install samba

Langkah 5

Jika installasi telah selesai, kita akan mulai mengkonfigurasi file server/samba tersebut dengan menggunakan perintah nano /etc/samba/smb.conf

Langkah 6

Setelah itu tambahkan syntax dibawah ini pada baris paling bawah pada file smb.conf

Langkah 7

Jika sudah save dengan menekan Ctrl+x-y-ENTER

Langkah 8

Buat user untuk samba sesuai dengan nama user yang telah kita masukan tadi gunakan perintah

adduser user1

adduser user2

adduser user3

Langkah 9

Setelah itu kita akan mendaftarkan user yang baru kita buat ke samba dengan menggunakan perintah

smbpasswd -a user1

smbpasswd -a user2

smbpasswd -a user3

Langkah 10

Selanjutnya kita akan merestart network dan file server/samba dengan cara

service networking restart

/etc/init.d/samba restart

Langkah 11

Kita bisa melakukan pengujian konfigurasi yang kita lakukan berhasil atau tidak dengan menggunakan perintah testparm

Langkah 12

Sebelum melakukan pengujian pastikan antara server dan client saling terhubung, selanjutnya buka aplikasi run pada komputer/laptop lalu ketikan ip server masing-masing 192.168.0.2 (IP Server)

Langkah 13

Proses installasi dan konfigurasi file server/samba pun telah selesai
