UPDATE REPOSITORY OFFLINE PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Ketikan perintah apt-cdrom add untuk mengindex iso dvd debian 1-3

Langkah 3

Setelah selesai mengindex ketiga iso/dvd debian tersebut. selanjutnya ketikan perintah nano /etc/apt/sources.list untuk melihat hasil index iso yang sebelumnya kita index

Langkah 4

Setelah itu berikan tanda # pada repository online, seperti pada gambar dibawah ini

Langkah 5

Lalu save dengan menekan Ctrl+x-y-ENTER

Langkah 6

Setelah itu kita akan mengupdate repository offline dengan menggunakan perintah apt-get update tunggu hingga update selesai

Langkah 7

Update repository offline telah berhasil
