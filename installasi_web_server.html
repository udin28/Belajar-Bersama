INSTALLASI WEB SERVER ( APACHE ) PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Ketikan perintah apt-get install apache2 php5 phpmyadmin mysql-server untuk menginstall Apache2 , php5, phpmyadmin, mysql-server ikuti prosesnya sampai selesai

Langkah 3

Setelah selesai masuk ke direktory /etc/apache2/sites-available kemudian copy file 000-default menjadi web.conf selanjutnya buka web.conf

cd /etc/apache2/sites-available

cp 000-default.conf web.conf

nano web.conf

Langkah 4

Selanjutnya tambahkan syntax di bawah ini

Langkah 5

Lalu save dengan menekan Ctrl+x-y-ENTER

Langkah 6

Setelah itu masuk ke direktory /var/www kemudian buat folder web

cd /var/www

mkdir web

Langkah 7

Selanjutnya anda dapat memasukan halaman web yang anda miliki kedalam folder tersebut, sebenarna secara default sudah ada sebuah halaman web yang bernama index.htmldi dalam folder html. kali ini kita akan membuat sebuah file index.html pada folder web yang telah kita buat tadi

cd web

nano index.html

Langkah 8

Setelah itu anda dapat mengedit file index.html sesui keinginan anda, contohnya seperti di bawah ini

Langkah 9

Jika sudah kita akan mematikan website 000-default.conf dan mengaktifkan konfigurasi website kita tadi yaitu web.conf

a2dissite 000-default.conf

a2ensite web.conf

Langkah 10

Restart network dan apache2 dengan cara

service networking retart

service apache2 restart atau /etc/init.d/apache2 restart

Langkah 11

Tes apakah web server sudah berjalan pada server debian dengan cara

w3m domain.com / w3m 192.168.0.2 jika sudah muncul halaman webnya maka web server telah berhasil

Langkah 12

Masuk ke browser kemudian di address bar ketikan alamat domain / alamat ip kita

Langkah 13

Proses installasi dan konfigurasi web server telah selesai
