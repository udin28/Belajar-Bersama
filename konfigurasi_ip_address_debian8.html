KONFIGURASI IP ADDRESS PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Ketikan perintah nano /etc/network/interfaces untuk konfigurasi ip address.

Langkah 3

Ubah ip address menjadi seperti gambar di bawah ini

Langkah 4

Setelah semua di ubah lalu save dengan menekan Ctrl+c-y-ENTER

Langkah 5

Lalu restart network dengan mengetikan perintah /etc/init.d/networking restart

atau

Menggunakan perintah service networking restart

Langkah 6

Periksa apakah konfigurasi ip itu berhasil atau tidak dengan memasukan perintah ifconfig jika berhasil maka akan seperti ini

Langkah 7

Jangan lupa untuk mengganti ip komputer/laptop yang kita gunakan menjadi ip dengan segmen yang sama dengan yang tadi kita konfigurasi pada linux debian

Seperti gambar di bawah ini

Langkah 8

Setelah ip pada komputer/laptop kita sudah di ubah. Kita bisa test apakah konfigurasi itu berhasil atau tidak

Dengan memasukan perintah ping 192.168.0.2 dan ping 192.168.0.1 jika berhasil maka akan seperti ini

Langkah 9

Selesai! Sekarang IP address pada Debian 8 telah dikonfigurasi.
