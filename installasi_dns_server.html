INSTALLASI DNS SERVER ( BIND9 ) PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Untuk mengaktifkan fitur DNS server kita membutuhkan sebuah aplikasi yang bernama bind9, untuk menginstall bind9 gunakan perintah apt-get install bind9 ikuti prosesnya hingga selesai

Langkah 3

Setelah selesai menginstall bind9 selanjutnya ketikan perintah cd /etc/bind untuk masuk kedalam direktory bind

Langkah 4

Kemudian copy file forward ( untuk mengkonfersi nama domain ke alamat ip ) dan reverse ( mengkonfersi alamat ip ke nama domain ) dengan cara

cp db.local db.f-dns ( penulisan nama untuk db.f-dns bisa disesuaikan keinginan kalian )

cp db.local db.r-dns ( penulisan nama untuk db.r-dns bisa disesuaikan keinginan kalian )

Langkah 5

Lalu masuk ke file db.f-dns dengan menggunakan perintah nano db.f-dns ubah semua tulisan localhost menjadi nama domain kita ( disini saya menggunakan alamat domain udin.com ) kemudian ubah ip yang ada di bawah menjadi ip kita. pada baris terakhir, symbol '@' di ganti dengan 'www', lalu 'AAA' diganti 'A'

Langkah 6

Setelah itu kita masuk juga kedalam file db.r-dns dengan menggunakan perintah nano db.r-dns ubah semua tulisan localhost menjadi nama domain kita, kemudian tambahkan 2 oktet terakhir dari ip kita yang di tulis secara terbalik (dari kanan)

Langkah 7

Masuk kedalam file named.conf dengan menggunakan perintah nano named.conf kemudian tambahkan syntax di bawah ini

Langkah 8

Lalu setelah itu masuk ke file resolv.conf nano resolv.conf untuk melihat apakah nama domain dan ip kita sudah ada atau belum

Langkah 9

Kemudian restart network dan bind9 dengan cara

service networking restart

service bind9 restart

Langkah 10

Untuk pengujian gunakan perintah nslookup 192.168.0.2 dan nslookup domain.com

Langkah 11

Installasi dan konfigurasi DNS server telah selesai
