INSTALLASI SSH SERVER PADA LINUX DEBIAN 8

Langkah 1

Pastikan login menggunakan akun super user

Langkah 2

Sebelum menginstall ssh server pastikan bahwa kalian telah mengupdate repository terlebih dahulu, untuk melihat apakah kalian sudah update atau belum ketikan perintah nano /etc/apt/sources.list jika masih kosong silahkan update repository terlebih dahulu

Langkah 3

Untuk menginstall SSH server kita gunakan perintah apt-get install openssh-server ikuti prosesnya sampai installasi selesai

Langkah 4

Setelah itu kita akan mengganti port pada SSH tersebut, dengan mengetikan perintah nano /etc/ssh/sshd_config lalu ubah port dengan angka berapapun yang kalian inginkan

Langkah 5

Lalu setelah itu cari kalimat PermitRootLogin. gunakan Ctrl+w lalu ketikan PermitRootLogin, setelah itu ubah kata without-password menjadi yes

Langkah 6

Setelah itu kita akan merestart network dan ssh dengan cara

service networking restart

service ssh restart

Langkah 7

Setelah di restart kita akan melakukan pengujian dengan menggunakan aplikasi putty

Langkah 8

Buka aplikasi putty, lalu pada menu hostname masukan ip contoh 192.168.0.2 pada menu port ubah port default ke port yang telah di konfigurasi tadi

Langkah 9

Jika berhasil maka akan seperti gambar di bawah ini, kalian bisa login menggunakan username dan password kalian masing-masing

Langkah 10

Proses installasi dan konfigurasi SSH server telah selesai
